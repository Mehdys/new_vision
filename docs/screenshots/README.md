# Screenshots

This folder contains screenshots of the NewVision application.

## How to Add Screenshots

1. **Take screenshots** of your application:
   - Home page (`home.png`)
   - Feedback form (`feedback-form.png`)
   - Dashboard (`dashboard.png`)
   - Embed widget in action (`embed-widget.png`)

2. **Save the images** in this folder with the names above (or update the README.md to match your filenames)

3. **Recommended image format**: PNG or JPG
   - **Recommended size**: 1200-1920px width
   - **File size**: Keep under 500KB per image for fast loading

4. **Commit and push** to GitHub:
   ```bash
   git add docs/screenshots/
   git commit -m "Add screenshots"
   git push
   ```

## Screenshot Ideas

- Home page with hero section
- Feedback form modal/widget
- Dashboard with feedback list
- Embed widget on an external website
- Mobile responsive views
- Dark/light theme variations
